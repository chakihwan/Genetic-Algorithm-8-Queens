# 8-Queens 유전자 알고리즘 구현 (Genetic Algorithm)

이 프로젝트는 고전적인 조합 최적화 문제인 **8-Queens 문제**를 **유전자 알고리즘(Genetic Algorithm)**을 사용하여 해결합니다. 8x8 체스판에 8개의 퀸(Queen)을 서로 공격할 수 없는 위치(같은 행, 열, 또는 대각선)에 배치하는 최적해를 찾는 것을 목표로 합니다.



## 1. 프로젝트 접근 방식: 유전자 알고리즘

이 문제는 유전자 알고리즘의 핵심 요소를 적용하여 해결되었습니다. `Genetic_2.py` 스크립트는 이 모든 과정을 구현하고 있습니다.



---

## 2. 알고리즘 핵심 구현 (`Genetic_2.py`)

`Genetic_2.py` 스크립트는 `Chromosome` 클래스와 메인 실행 루프를 통해 GA의 핵심 과정을 구현합니다.

### 1. 염색체 (Chromosome)

* **정의**: 과제 보고서의 정의에 따라, 하나의 해(염색체)는 **8개의 유전자(1~8 사이의 정수)를 가진 리스트**로 표현됩니다.
* **의미**: 리스트의 **인덱스**는 체스판의 **열(Column)**을, 해당 인덱스의 **값**은 **행(Row)**을 의미합니다.
    * 예: `[7, 3, 8, 2, 5, 1, 6, 4]`

### 2. 적합도 함수 (Fitness Function)

* **목표**: 염색체(해)가 얼마나 "좋은지"를 평가합니다.
* **구현**: `calculate_fitness()` 함수에서 구현됩니다.
* **계산**: 8개의 퀸 중에서 2개를 뽑는 모든 조합(C(8, 2) = 28가지)에 대해 서로 공격하지 않는(행, 대각선이 겹치지 않는) 쌍의 개수를 셉니다.
* **최적해**: 모든 퀸이 서로 공격하지 않는 상태이며, **최대 적합도는 28**입니다. ( `TARGET_FITNESS = 28` )

### 3. 진화 과정 (Selection, Crossover, Mutation)

알고리즘은 `TARGET_FITNESS` (28)에 도달할 때까지 다음 과정을 반복합니다.

1.  **선택 (Selection)**: 현재 세대(`population`)를 적합도 순으로 정렬하고, 가장 우수한 **상위 2개의 염색체**(`parent1`, `parent2`)를 부모로 선택합니다.
2.  **교차 (Crossover)**: `crossover` 함수가 두 부모의 유전자를 조합하여(Single-point crossover) 자식 염색체(`child`)를 생성합니다.
3.  **변이 (Mutation)**: `mutate` 함수가 `MUTATION_RATE` (10%) 확률로 자식의 유전자 2개를 무작위로 교체(swap)하여 돌연변이를 일으킵니다.
4.  **세대 교체 (Elitism)**: **엘리트주의**를 적용하여, 생성된 자식(child)이 현재 세대에서 **가장 적합도가 낮은 염색체 1개를 대체**합니다.
